# 群聊刺客 OlivOSAIChatAssassin

## 写在前面

**群聊刺客 - 论社交的最小系统与图灵测试的终结**

原教旨主义图灵测试通常设定为一位人类裁判明确知道有一个机器并试图从诸多一对一场景中识别它，但这对当今的互联网现实而言是否太狭隘而不具有实际意义了？

当我意识到这一点时，我便开始观察网友们的行为模式。最终，我得出了一个略带讽刺却符合直觉的结论：在充斥着碎片化信息的互联网浅社交中，大多数人的发言呈现出 “低信息密度、低组织成本、低参与意识与高行为重复” 的特征。在这样的语境下，人们分辨对话者是AI还是人类的警惕性本就极为淡薄，甚至可以说是主动放弃了鉴别。在这种低质量的信息交互中，AI甚至不需要刻意隐藏，因为它输出的内容与人类输出的内容，已经没有显著区别——无论是在信息熵上，还是价值观上。

这意味着，我完全可以转变思路：不再以制造人类的通用仿品为目标去搭建一个AI Agent，而是将目标限定在“多人即时通讯群聊”这一特定场景中——一个账号信息没有特别标注、且言行与设定高度一致的AI账号。

虽然，对于那些将互联网浅社交视为水与空气的人而言这有些冒犯，但……

不妨设想一下：若是你在每天水群的群聊中，看到一个顶着常见二次元或抽象文化头像、用着大众脸昵称的群友，在你说话时时不时附和几句，有时反对你几句——你或许根本就不会多想。这个项目甚至不需要有大语言模型的参与。相比之下，大语言模型只是极大拓展了其上限，并略微抬高了其下限，而或许，也已远远超过其真正所需。

正如一千个读者有一千个哈姆雷特，但AI不需要逐一实现那一千个平行世界中的希斯克里夫——它只需要人类以为它是“某个世界的凯瑟琳”，这就足够了，因为，在人际交互的最小系统里，存在感可以是一种集体幻觉。

所以我写了一个OlivOS插件，并在过去的一段时间内于几十余个群内开展了测试。

从结论上来说，实验结果符合预期。它指向了未来AI与人类共存的常态：AI不会以完美的形态出现——因为它不需要；它悄无声息地融入我们的日常场景，重塑我们的社交习惯。

这或许预示着，真正的图灵测试已经变成了一个社会学问题，而非技术问题。AI试着证明自己是人已经成为了过去式。人是否还愿意、以及能够表现得不像AI——这才是需要付出额外努力的事情。

或许从此以后，一切都将改变。

## 用法
首次运行后会在`plugin/data/OlivOSAIChatAssassin`目录下释放`config.json`和`memory.json`两个文件，文件基本结构如下。

### config.json
```json
{
    "api_key": "",
    "api_base": "https://api.deepseek.com/v1",
    "model": "deepseek-chat",
    "max_tokens": 512,
    "temperature": 0.7,
    "personality": "你是一个伪装成人类的自豪的新锐AI，你择机加入聊天，避免过多尬聊。别让他们发现你的真实身份！！！",
    "enabled_groups": [
        "all"
    ],
    "history_size": 10,
    "reply_probability": 1,
    "reply_keywords": [],
    "mention_reply": true,
    "ignore_prefixes": [],
    "max_message_length": 2000
}
```
- 你需要修改`api_key`为你自己的
- 修改`api_base`为你选用的`Open AI 兼容`接口
- `enabled_groups`为支持的群组列表，字符串格式，填写`all`则在所有群开启
- `history_size`前文消息条数，修改后需要重载插件生效

### memory.json
```json
{
    "全局": {
        "常识": [],
        "知识搜索": {},
        "人物关系": {}
    }
}
```
- 此文件会被插件实时更新，其中`全局`条目下的内容可以由用户自行添加，或者通过额外脚本进行定期更新。
